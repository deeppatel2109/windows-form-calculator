# Simple Calculator

A straightforward, easy-to-use desktop calculator built with C# and Windows Forms. I created this project to practice C# desktop application development.

## Features

- Basic arithmetic operations (addition, subtraction, multiplication, division)
- Clean, intuitive graphical user interface
- Fast and responsive

## Getting Started

### Prerequisites
- Windows OS
- Visual Studio (or any compatible IDE for C# WinForms)
- .NET Framework / .NET 

### Running the App
1. Clone or download this repository.
2. Open the `Calculator.csproj` file in Visual Studio.
3. Build the solution.
4. Click **Start** to run the application.

## Built With

- C#
- Windows Forms (WinForms)
